Open Source & Free

Manage Crontab
Visually

A lightweight, beautiful desktop app for managing your local crontab jobs. No Docker, no Node.js server — just install and use.

Download GitHub
🛠

Visual CRUD

Create, edit, delete, and toggle cron jobs with a clean UI. No more crontab -e.

📅

Cron Expression Builder

Quick presets, real-time validation, human-readable description, and next 5 execution previews.

⚡

Run & Monitor

Execute any job instantly and see stdout, stderr, exit code, and duration in real time.

🔄

Import & Sync

One-click import from existing crontab. Changes sync to system crontab automatically.

🛡

Command Validation

Checks if the executable exists and warns about dangerous commands before saving.

🌐

Bilingual UI

Full Chinese and English support. Switch languages instantly from the header.

🎨

Dark Mode

Light, dark, and system-follow themes. Easy on the eyes at any hour.

🚀

Auto Update

Built-in updater checks for new versions on startup. One click to download and install.

📊

Dashboard

Overview of total jobs, active count, recent executions, and failure alerts at a glance.

Built With

Tauri v2 Rust React 19 TypeScript Vite 6 Tailwind CSS v4 SQLite
